# 🚀 CareerHub - Modern Job Portal
Built for **CodSoft Level 2 Task 1**.

### ✨ Features:
- **Professional Landing Page:** Modern hero section with a clean purple theme.
- **Employer Dashboard:** Dedicated area to post new job opportunities and manage entries.
- **Real-time Integration:** Full-stack connectivity using MERN (MongoDB, Express, React, Node).
- **Modern UI/UX:** Clean navigation, responsive design, and professional icons.

### 🛠️ Tech Stack:
- **Frontend:** React.js, CSS3 (Custom Montserrat font)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **API Handling:** Axios

### 📦 How to Run:
1. **Server:** Go to `/server`, run `npm install` then `npm start`.
2. **Frontend:** Go to `/frontend-app`, run `npm install` then `npm start`.
